Kebijakan Privasi - DomKa

DomKa adalah aplikasi pencatatan keuangan offline yang tidak mengumpulkan, menyimpan, membagikan, atau memproses data pribadi pengguna.

1. Pengumpulan Data

Aplikasi ini tidak mengumpulkan data pribadi apa pun. Semua data seperti catatan transaksi, pemasukan, dan pengeluaran disimpan secara lokal di perangkat pengguna.

2. Penggunaan Data

Data hanya digunakan oleh pengguna sendiri untuk melihat riwayat dan laporan keuangan. Tidak ada data yang dikirimkan ke server mana pun.

3. Keamanan

Karena seluruh data disimpan secara lokal, keamanan data bergantung pada keamanan perangkat pengguna.

4. Izin (Permissions)

Aplikasi DomKa tidak meminta izin apa pun selain yang diperlukan untuk fungsi dasar aplikasi.

5. Perubahan Kebijakan

Kebijakan ini dapat diperbarui sewaktu-waktu. Pengguna disarankan untuk memeriksa halaman ini secara berkala.
